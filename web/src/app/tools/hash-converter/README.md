# ハッシュ変換

入力した文字列を複数のハッシュ値に変換するツール。

## 機能

- **複数ハッシュの一括計算** — 入力文字列から複数アルゴリズムのハッシュ値を生成
- **UTF-8 入力** — 入力文字列を UTF-8 のバイト列として処理
- **コピー** — 各ハッシュ値をクリップボードにコピー
- **ブラウザ完結** — 入力値はサーバーに送信されず、ブラウザ内でのみ処理

## 対応ハッシュ

- MD5
- SHA-1
- SHA-256
- SHA-384
- SHA-512
- SHA3-256
- SHA3-512
- Keccak-256
- RIPEMD-160
- BLAKE3

## 注意

- MD5 と SHA-1 は衝突耐性が弱いため、新しい用途では推奨されません
- 署名や認証用途では、用途に合ったアルゴリズムと手順を選んでください
