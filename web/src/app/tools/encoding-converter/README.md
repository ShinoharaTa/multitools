# 文字コード変換

入力文字列を指定文字コードのバイト列にし、そのバイト列を別の文字コードとして読み直すツール。

## 機能

- **文字列 -> バイト列 -> 文字列** — 入力文字列を一度バイト列化し、別の文字コードとして再解釈
- **文字化け確認** — Shift_JIS を UTF-8 として読むなど、誤った解釈時の見え方を確認
- **複数形式の表示** — 中間バイト列を hex、`%XX`、Base64 で表示
- **コピー** — 各形式の変換結果をクリップボードにコピー

## 対応文字コード

- UTF-8
- Shift_JIS
- EUC-JP

## 使い方

1. 入力文字列をバイト列にする文字コードを選ぶ
2. そのバイト列を読み直す文字コードを選ぶ
3. 入力欄に文字列を入力する
4. 読み直した文字列や中間バイト列を確認・コピーする
