# IPアドレス確認

現在のアクセス元として見えているグローバルIPアドレスを表示するツール。

## 機能

- **IP表示** — 外部の IP 確認 API からグローバルIPアドレスを取得
- **コピー** — 表示されたIPアドレスをクリップボードにコピー
- **再取得** — 現在の通信経路で見えているIPアドレスを再取得

## 注意

- 表示されるのは LAN 内のプライベートIPではなく、外部サイトから見えるグローバルIP
- VPN、プロキシ、モバイル回線、Cloudflare WARP などを使っている場合は、その出口IPが表示される
- IP取得には外部APIへの通信が必要
