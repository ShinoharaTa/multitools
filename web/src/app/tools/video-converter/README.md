# Video Converter

動画のアスペクト比を修正し、安定したフォーマットに変換するツール。変換処理は ffmpeg.wasm によりブラウザ内で完結する。

## 機能

- **動画情報の表示** — 解像度、アスペクト比、長さをブラウザで解析して表示
- **アスペクト比の変更** — プリセット (16:9, 4:3, 1:1, 9:16, 21:9) またはカスタム値
- **安定フォーマットへの変換** — H.264 + AAC / MP4 に統一し、音ずれを防止
- **ブラウザ完結** — サーバーへのアップロード不要。ファイルは端末内でのみ処理される

## 変換仕様

| 項目 | 設定 |
|------|------|
| 映像コーデック | H.264 (libx264) |
| 映像品質 | CRF 18 |
| ピクセルフォーマット | yuv420p |
| 音声コーデック | AAC |
| 音声ビットレート | 192kbps |
| サンプルレート | 48kHz ステレオ |
| フレームレート | CFR (固定) に変換 |
| コンテナ | MP4 (faststart) |

## 音ずれ対策

- `-fps_mode cfr`: 可変フレームレート (VFR) を固定フレームレート (CFR) に変換
- AAC エンコード: 互換性の高い音声コーデックに統一

## 仕組み

- [@ffmpeg/ffmpeg](https://www.npmjs.com/package/@ffmpeg/ffmpeg) の単一スレッド版を使用
- ffmpeg コア (約30MB) は初回変換時に CDN から遅延ロード
- `SharedArrayBuffer` を使わないため COOP/COEP ヘッダーは不要

## 制約

- 初回はコア読み込みのため起動が重い
- 変換速度はネイティブ ffmpeg より遅い
- 大きい動画はブラウザのメモリ制限で失敗する場合がある
