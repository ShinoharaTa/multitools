# Video Converter

動画のアスペクト比を確認・修正し、安定したフォーマットに変換するツール。

## 機能

- **動画情報の解析** — コーデック、解像度、アスペクト比 (DAR/SAR)、FPS、音声情報を表示
- **問題の自動検出** — 非正方ピクセル(SAR)、非標準コーデック、可変フレームレートを警告
- **アスペクト比の変更** — プリセット (16:9, 4:3, 1:1, 9:16, 21:9) またはカスタム値
- **安定フォーマットへの変換** — H.264 + AAC / MP4 に統一し、音ずれを防止

## 変換仕様

| 項目 | 設定 |
|------|------|
| 映像コーデック | H.264 (libx264) |
| 映像品質 | CRF 18 |
| ピクセルフォーマット | yuv420p |
| 音声コーデック | AAC |
| 音声ビットレート | 192kbps |
| サンプルレート | 48kHz ステレオ |
| フレームレート | CFR (固定) に変換 |
| コンテナ | MP4 (faststart) |

## 音ずれ対策

- `-vsync cfr`: 可変フレームレート (VFR) を固定フレームレート (CFR) に変換
- `-async 1`: 音声ストリームのタイムスタンプを映像に同期
- AAC エンコード: 互換性の高い音声コーデックに統一

## 必要環境

- ffmpeg / ffprobe がインストール済みであること
